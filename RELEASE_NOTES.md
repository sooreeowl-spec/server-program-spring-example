# Release Notes

## v1.1.0

### 주요 변경 사항
- 커뮤니티 화면 좌측 계정 영역 UI 개선
  - 기존 회원가입 입력 박스를 제거하고 **회원가입 버튼만 유지**
  - 회원가입 버튼 클릭 시 **모달(팝업) 회원가입 창**으로 입력/가입 처리
  - 기존 회원가입 버튼의 **색상(초록 계열) 유지**
- 게시판 서비스명 변경
  - `Koreait 커뮤니티 게시판` → **`CodeBridge 커뮤니티 게시판`**

### 주요 기능
- 세션 기반 로그인/로그아웃
- 회원가입, 사용자 정보 조회
- 게시글 CRUD + 페이징
- 댓글 CRUD
- 공통 응답/예외 처리 및 보안(401/403)

### 실행 방법 (간단)
```bash
./gradlew bootRun
```

브라우저 접속:
- `http://localhost:8080/community.html`
